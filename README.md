# react-jwt-api-adapter

> React context provider for requesting API&#x27;s with expiring JWT&#x27;s.

[![NPM](https://img.shields.io/npm/v/react-jwt-api-adapter.svg)](https://www.npmjs.com/package/react-jwt-api-adapter) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save react-jwt-api-adapter
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-jwt-api-adapter'
import 'react-jwt-api-adapter/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
